Learn security by doing — no VM, no setup pain.

BEFORE YOU BEGIN

Module 00 — Set up Docker (once)

Install Docker Desktop, clone the repo, and run ./start.sh. That's the only prerequisite — every module below is independent and can be done in any order.

git clone github.com/michael-borck/assume-breach-labs cd assume-breach-labs && ./start.sh
